# lily58-wireless-view-zmk-config

Modified the default keymaps to suit a basic "Windows" office work environment. Rev 1 is a first draft and will likely change multiple times in the future.

Keymaps modified using visual online keymap editor by Nick Coutsos https://github.com/nickcoutsos/keymap-editor
(images below show that revisions layout as seen in the keymap editor)

*comments in the keymap file do not match the current keymap, those are from the original author's defualts.
